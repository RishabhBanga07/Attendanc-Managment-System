# 📌 Attendance Management System (Java)

## 📖 Overview
The **Attendance Management System** is a Java-based application designed to efficiently manage and track student attendance. It replaces manual record-keeping with a simple digital solution, making attendance tracking faster, organized, and reliable.

This project is built using core Java and demonstrates strong use of Object-Oriented Programming (OOP) concepts like encapsulation, modularity, and data abstraction. It is ideal for beginners and academic use.

---

## 🚀 Features
- ➕ Add new students  
- 📋 View student list  
- ✅ Mark students as Present  
- ❌ Mark students as Absent  
- 📊 Track attendance records  
- 💾 Save and load data (if file handling is used)  

---

## 🛠️ Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- File Handling (optional)
- Collections Framework (ArrayList / Vector)

## ⚙️ How It Works
1. Run the program  
2. A menu is displayed with options:
   - Add Student  
   - Mark Attendance  
   - View Records  
3. Students are stored in a list  
4. Attendance is updated based on input  
5. Data can be saved and reused
